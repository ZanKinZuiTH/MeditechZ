# คู่มือผู้ใช้: การแก้ไขรายละเอียดการตรวจ X‑Ray (RIS → PACS WorkList)

## ขั้นตอนใช้งาน
1) เปิด RIS → PACS WorkList เลือกแถวที่ต้องการ
2) คลิกปุ่ม “แก้ไขรายละเอียด”
3) ปรับค่า: อวัยวะที่ตรวจ, รายละเอียดการตรวจ, Modality, หมายเหตุผู้ป่วย
4) ตรวจสรุปการเปลี่ยนแปลง (ภาษาไทย)
5) กดยืนยัน “บันทึก” → ระบบบันทึกและสร้างประวัติ Audit
6) กด “รีเฟรชประวัติ” เพื่อดูรายการล่าสุด

## เงื่อนไขการใช้งาน
- สิทธิ์บทบาทที่อนุญาต: AdminRadiologist, Radiologist, RDUStaff, Admin
- Modality ที่อนุญาต: CR, DX, CT, ES, MR, MG, OT, RF, US
- ข้อจำกัดความยาว: StudyDescription ≤ 256, BodyPartsInStudy ≤ 512, ModalitiesInStudy ≤ 256, PatientComments ≤ 4000

## คำแนะนำ
- หากเปิดใช้ mapping Bodypart ค่าบางอย่าง (เช่น CXR) จะถูก map เป็นค่ามาตรฐาน (เช่น CHEST)
- หากบันทึกไม่สำเร็จ ให้ตรวจสอบข้อความเตือนบนหน้าจอ และสิทธิ์บทบาทผู้ใช้
